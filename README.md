# MicroIP

Greenfield Micro Suite command-line utility.

## Install

```bash
pip install microip
```

## Run

```bash
microip
```

## License

MIT

---

## Overview

Simple utility for displaying and understanding IPv4 addressing information.

## Features

- Lightweight command-line interface
- Cross-platform Python
- Educational design
- Minimal dependencies
- Fast execution

## Educational Concepts

- IPv4 addressing
- Public vs private IPs
- Network interfaces
- Address structure
